Recent Projects

Project 1: Unveiling Patterns in French Weather and Electricity Consumption

This data analysis project delves into the intricate relationship between historical daily weather data for France and its correlation with electricity consumption. Navigating through ten detailed steps, the analysis includes data retrieval, correlation matrix calculations, synchronization of time series, and the application of multivariate regression models.

Project 2: Credit Card Segmentation and Prediction

In this report, I present a comprehensive analysis of a Credit Card Segmentation and Prediction project. The context involves utilizing machine learning techniques to classify credit card users into distinct segments. The purpose is to develop a predictive model for segmenting credit card users based on their transactional patterns.

Project 3: Deep Analysis in Anomaly-Based Intrusion Detection and Prevention

In this personal data science endeavor, I undertook a comprehensive analysis of the CICIDS2017 cybersecurity dataset, delving deep into the intricacies of Anomaly-Based Intrusion Detection and Prevention. This project aimed to enhance my data science proficiency by extracting meaningful insights from network traffic data and providing valuable perspectives for cybersecurity strategies.

Project 4: Advanced Image Classification using Deep Learning Techniques

This project focuses on leveraging deep learning algorithms to perform advanced image classification tasks. The work involved preprocessing images, tuning hyperparameters, and deploying the model for real-world applications in healthcare and autonomous systems.
